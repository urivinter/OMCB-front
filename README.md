# One Million Checkboxes (Front End)

This is the front-end for the One Million Checkboxes project. It is a React application built with Vite.

## Local Installation and Setup

To run this project locally, follow these steps:

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/urivinter/OMCB-front
    cd OMCB-front
     ```

2.  **Install dependencies:**

    Make sure you have Node.js and npm installed. Then, run the following command in the `front` directory to install the required packages:

    ```bash
    npm install
    ```

3.  **Run the development server:**

    To start the local development server, run:

    ```bash
    npm run dev
    ```

    This will start the application on a local server, typically at `http://localhost:5173`.
